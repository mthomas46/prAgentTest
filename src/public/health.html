Health Check Endpoints

This page provides information about the health check endpoints available in the API.

Basic Health Check

GET /api/health

Returns basic health status information.

Response

{
    "status": "ok",
    "timestamp": "2024-04-02T08:00:00.000Z",
    "uptime": 3600,
    "memoryUsage": {
        "heapUsed": 123456789,
        "heapTotal": 987654321,
        "external": 123456789
    }
}

Detailed Health Check

GET /api/health/detailed

Returns detailed health information including memory usage statistics.

Response

{
    "status": "ok",
    "timestamp": "2024-04-02T08:00:00.000Z",
    "uptime": 3600,
    "memoryUsage": {
        "heapUsed": 123456789,
        "heapTotal": 987654321,
        "external": 123456789,
        "rss": 123456789
    },
    "process": {
        "pid": 12345,
        "version": "v18.0.0",
        "platform": "darwin",
        "arch": "x64"
    }
}

Response Fields

Field Type Description
status string Current health status ("ok" or "error")
timestamp string ISO timestamp of the health check
uptime number Server uptime in seconds
memoryUsage object Memory usage statistics
process object Process information (detailed endpoint only)
